# employee-management-sql
Designed and implemented a relational SQL Server database with ER modeling, triggers, views, stored procedures, roles, and backup management using SSMS.

## Employee Management Database (SQL Server)

### Project Overview
This project involves designing and implementing a relational database
for an automobile company using SQL Server.

The database supports efficient employee, customer, order, and payment
management with proper relational integrity.
